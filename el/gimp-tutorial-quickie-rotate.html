4.8. Περιστροφή εικόνας

Εικόνα 3.39. Μενού για “περιστροφή εικόνας”


Ας υποθέσουμε ότι περιστρέφετε την ολοκαίνουργια ψηφιακή σας κάμερα για να κάνετε μια κάθετη λήψη και τώρα μερικές από τις εικόνες σας είναι πλάγιες. Ενεργοποιείστε Εικόνα → Μετασχηματισμός → Περιστροφή 90° δεξιόστροφα. Χρησιμoποιώντας μια εικόνα APOD. Την περιέστρεψα μια φορά για να δείτε 90 μοίρες αριστερόστροφα.
