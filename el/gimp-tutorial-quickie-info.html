4.5. Εύρεση πληροφοριών για την εικόνα σας

Εικόνα 3.29. Εύρεση πληροφοριών


Αυτό το παράδειγμα χρησιμοποιεί μια άλλη εικόνα από το APOD [APOD06]. Πολλές πληροφορίες είναι διαθέσιμες άμεσα από το παράθυρο εικόνας, π.χ. οι διαστάσεις της εικόνας. Ενεργοποιείστε Εικόνα → Ιδιότητες εικόνας για να ανοίξετε το διάλογο ιδιότητες εικόνας, που περιέχει ακόμα περισσότερες πληροφορίες.

Εικόνα 3.30. Διάλογος ιδιοτήτων εικόνας


Εάν κάνετε ένα τετράγωνο από το ορθογώνιο, όπως στο Ενότητα 4.4, “Περικοπή εικόνας”, είναι γρήγορος και εύκολος τρόπος για να βρείτε τη μικρότερη διάσταση. Χρειαζόσαστε μόνο να ανοίξετε το διάλογο και να βρείτε το πιο μικρό μήκος και να το χρησιμοποιήσετε όπως περιγράφηκε. Επειδή αυτό είναι πολύ μικρή πληροφορία και σίγουρα όχι αρκετή για να γεμίσετε το χώρο του εικονιδίου του μενού και του διαλόγου στιγμιότυπου στη διάταξή μου, σκέφτηκα να μερικούς υπολογισμούς που ίσως σας βοηθήσουν να βρείτε τις ανάγκες της εικόνα σας.

Εικόνα 3.31. Προβλήματα κλίμακας


Είναι ωραίο να εμφανίζονται εικόνες στο παράθυρο του περιηγητή όπως οι φωτογραφίες. Οι φωτογραφίες εμφανίζονται να είναι 4x6 ίντσες όταν προσαρμόζονται στα 288x432 εικονοστοιχεία (72 dpi για πολλές οθόνες). Υπάρχει, όμως, ένα πρόβλημα, εάν προσπαθήσετε να κλιμακώσετε αυτή την εικόνα. Η αναλογία του πλάτους προς το μήκος της αρχικής εικόνας δεν ταιριάζει την αναλογία της φωτογραφίας. Έτσι για να κάνετε την κλιμακωμένη εικόνα στο σωστό μέγεθος μερικά εικονοστοιχεία πρέπει να περικοπούν από το ύψος. Η τελική εικόνα θα “εμφανίζεται” ως μία φωτογραφία 6x4 ίντσες σε πολλές οθόνες.

Εικόνα 3.32. Προβλήματα που επιλύονται με περικοπή


Υπάρχουν προβλήματα όταν αναμιγνύονται σαρωμένες εικόνες, ψηφιακές φωτογραφίες και σαρωμένα αρνητικά. Πολλές μηχανές επεξεργασίας φιλμ περικόπτουν τμήματα της εικόνας αυτόματα έτσι ώστε να ταιριάζουν στο εκτυπώσιμο μέγεθος ή σε συγκεκριμένο στιλ. Εάν προετοιμάζετε μια εικόνα για εκτύπωση σε μία μηχανή όπως αυτή, ή εάν προετοιμάζετε μια συλλογή, όπου οι εικόνες είναι από διαφορετικές πηγές, μερική έξυπνη περικοπή για να ταιριάσει στο άριστο μέγεθος για το μέσο που επιλέξατε θα είναι ένα πλεονέκτημα. Εάν αυτό προκαλεί σύγχυση, κατηγορήστε τη βιομηχανία εκτύπωσης φωτογραφιών και όχι το GIMP.

Μπορείτε να αλλάξετε την ανάλυση της εικόνας σας επίσης, χρησιμοποιώντας τις ίδιες μεθόδους που χρησιμοποιήσαμε για κλιμάκωση της εικόνας, αν και στην κάπως περιορισμένη χρήση μου, το θέμα είναι περισσότερο στο πόσα εικονοστοιχεία. Ας πούμε ότι θέλετε να τυπώσετε μια φωτογραφία σε ένα εργαστήριο φωτογραφίας. Προτιμάται το 300 εικονοστοιχεία για κάθε ίντσα. Μια εικόνα που είναι 2241x1548 pixels θα εκτυπωθεί εύκολα ως μια φωτογραφία 7x5 ίντσες. 2241px/300ppi = 7,47 in. Κάντε τους υπολογισμούς για τη μικρή πλευρά. 1548/300 = _.

Υπάρχει επίσης ένα άλλο άσχημο γεγονός που πρέπει να λάβετε υπόψη, εάν είστε νέος στα γραφικά και στους υπολογιστές. Επειδή φαίνεται καλό στην οθόνη, δεν σημαίνει ότι θα τυπωθεί ωραία. Προσπάθησα να προσομοιώσω πώς αυτή η εικόνα θα εμφανίζεται τυπωμένη στα 300dpi. Συγνώμη. Υπάρχουν μερικές επιλογές, για παράδειγμα ένας φίλος μου τύπωσε εικόνες και μετά τις ξανασάρωσε. Απαίσια δουλειά!
