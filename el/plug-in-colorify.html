8.33. Colorify

8.33.1. Overview

Figure 15.176. Example for the “Colorify” filter

Original image

Filter “Colorify” applied


It renders a greyscaled image like it is seen through a colored glass. This filter works only on images in RGB mode.

8.33.2. Activate the filter

This filter is found in the image window menu under Colors → Colorify.

8.33.3. Options

Figure 15.177. “Colorify” filter options


Color

A color palette is available and you can select your own color by clicking on the Custom Color swatch.
