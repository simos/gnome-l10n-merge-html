2. Δημιουργία νέων αρχείων

Πατήστε Αρχείο → Νέο για να ανοίξετε διάλογο Δημιουργία νέας εικόνας. Τροποποιείστε το αρχικό πλάτος και ύψος του αρχείου ή χρησιμοποιείστε τις αρχικές τιμές, έπειτα δημιουργείστε ένα αρχείο νέας εικόνας. Περισσότερες πληροφορίες για το διάλογο Δημιουργία νέας εικόνας μπορούν να βρεθούν στο Ενότητα 2.2, “Νέο”.
