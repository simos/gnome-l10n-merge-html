3.2. Άνοιγμα τοποθεσίας

Εάν αντί για όνομα αρχείου, έχετε μια URI (π.χ. διεύθυνση ιστού) για την εικόνα, μπορείτε να την ανοίξετε χρησιμοποιώντας το μενού. Πατήστε Αρχείο → Άνοιγμα τοποθεσίας… από το μενού εικόνας. Αυτό σας φέρνει ένα μικρό διάλογο που σας επιτρέπει να εισάγετε ή να επικολλήσετε το URI.
