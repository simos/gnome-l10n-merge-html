5. Δημιουργία συντομεύσεων στις λειτουργίες μενού

Πολλές λειτουργίες που είναι προσπελάσιμες μέσα από το μενού της εικόνας έχουν μια προεπιλεγμένη συντόμευση πληκτρολογίου. Ίσως θέλετε να δημιουργήσετε μια νέα συντόμευση για μια εντολή που χρησιμοποιείται πολύ και δεν έχετε συντόμευση, ή πιο σπάνια, να επεξεργαστείτε μια υπάρχουσα συντόμευση. Υπάρχουν δύο μέθοδοι για αυτό.

Διαδικασία 11.1. Χρησιμοποιώντας δυναμικές συντομεύσεις πληκτρολογίου

  1. Πρώτα, πρέπει να ενεργοποιήσετε αυτή τη δυνατότητα διαλέγοντας την επιλογή Χρήση δυναμικών συντομεύσεων πληκτρολογίου στο στοιχείο Διεπαφή του μενού Προτιμήσεις. Αυτή η επιλογή δεν είναι συνήθως σημειωμένη, για την αποφυγή τυχαίων πατημάτων πληκτρολογίου που δημιουργούν ανεπιθύμητες συντομεύσεις.

  2. Ενώ κάνετε αυτό. επιλέξτε επίσης την επιλογή Αποθήκευση συντομεύσεων πληκτρολογίου κατά την έξοδο, έτσι η συντόμευση σας να αποθηκευτεί.

  3. Για να δημιουργήσετε μια συντόμευση πληκτρολογίου, απλά τοποθετήστε το δείκτη του ποντικιού σε μια εντολή του μενού: τότε θα επισημανθεί. Προσοχή ο δείκτης ποντικιού να μην κινείται και τυπώστε μια σειρά τριών πλήκτρων κρατώντας τα πλήκτρα πατημένα. Θα δείτε αυτή τη σειρά να εμφανίζεται στα δεξιά της εντολής.

  4. Είναι καλύτερο να χρησιμοποιήσετε τη σειρά Ctrl+Alt+πλήκτρο για τις προσαρμοσμένες συντομεύσεις.

Εικόνα 11.31. Ρύθμιση συντομεύσεων πληκτρολογίου


Διαδικασία 11.2. Χρήση του επεξεργαστή συντομεύσεων πληκτρολογίου

  1. Ανοίγετε αυτόν τον επεξεργαστή πατώντας στο Ρύθμιση συντομεύσεων πληκτρολογίου στο στοιχείο “Διεπαφή” του μενού Προτιμήσεις.

  2. Όπως φαίνεται σε αυτόν το διάλογο, μπορείτε να διαλέξετε την εντολή για την οποία θέλετε να δημιουργήσετε μια συντόμευση στην περιοχή “Ενέργεια”. Έπειτα τυπώνετε τη σειρά των πλήκτρων, όπως πιο πάνω. Το πλήκτρο διαστήματος θα έπρεπε να καθαρίσει μια συντόμευση (αλλά δεν τη διαγράφει.)

  3. Ο επεξεργαστής συντομεύσεων επίσης επιτρέπει τον έλεγχο των ρυθμίσεων των παραμέτρων του εργαλείου με το πληκτρολόγιο. Στην κορυφή αυτού του διαλόγου, μπορείτε να βρείτε ένα μενού περιεχομένων που σας οδηγεί στις παραμέτρους του εργαλείου. Για διευκόλυνση της εργασίας σας, οι τύποι εργαλείων σημειώνονται με μικρά εικονίδια.

Σημείωση

Οι προσαρμοσμένες συντομεύσεις πληκτρολογίου αποθηκεύονται σε έναν κρυμμένο κατάλογο του Gimp (/home/[username]/.gimp-2.6/menurc) στο Linux και C:\Documents and Settings\[Username]\.gimp-2.6\menurc στα Windows XP. Είναι ένα απλό αρχείο κειμένου που μπορείτε να μεταφέρετε από έναν υπολογιστή σε έναν άλλο.
