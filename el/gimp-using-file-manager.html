3.5. Διαχειριστής αρχείου

Εάν έχετε συνδέσει ένα τύπο αρχείου εικόνας με το GIMP, είτε όταν εγκαταστήσατε το GIMP είτε αργότερα, τότε μπορείτε να περιηγηθείτε στο αρχείο χρησιμοποιώντας ένα διαχειριστή αρχείου (όπως Nautilus ή Konqueror στο Linux, ή Windows Explorer στα Windows), και όταν το βρείτε, διπλοπατήστε το. Εάν τροποποιήθηκε κατάλληλα, η εικόνα θα ανοίξει στο GIMP.
