3. Δημιουργία νέων στρώσεων

Υπάρχουν πολλοί τρόποι δημιουργίας νέων στρώσεων σε μια εικόνα. Εδώ είναι οι πιο σημαντικοί:

  • Επιλέγοντας Στρώση → Νέα στρώση στο μενού εικόνας. Αυτό εμφανίζει ένα διάλογο που σας επιτρέπει να ορίσετε τις βασικές ιδιότητες της νέας στρώσης. Δείτε την ενότητα διάλογος νέας στρώσης για βοήθεια.

  • Επιλέγοντας Στρώση → Δημιουργία αντιγράφου στο μενού εικόνα. Αυτό δημιουργεί μια νέα στρώση, που είναι το τέλειο αντίγραφο της τρέχουσας ενεργής στρώσης, ακριβώς πάνω από την ενεργή στρώση.

  • Κάνοντας “Αποκοπή” ή “Αντιγραφή” και έπειτα επικόλληση χρησιμοποιώντας Ctrl+V ή Επεξεργασία → Επικόλληση, το αποτέλεσμα είναι μια “επιπλέουσα επιλογή”, που είναι ένα είδος προσωρινής στρώσης. Πριν να μπορείτε να κάνετε οτιδήποτε άλλο, πρέπει είτε να αγκυρώσετε την επιπλέουσα επιλογή σε μια υπάρχουσα στρώση, ή να την μετατρέψετε σε κανονική στρώση. Εάν κάνετε το δεύτερο, η νέα στρώση θα γίνει ακριβώς τόσο μεγάλη ώστε να περιέχει το επικολλημένο υλικό .
