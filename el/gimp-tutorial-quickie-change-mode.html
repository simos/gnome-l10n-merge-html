4.6. Αλλαγή κατάστασης

Όπως και οτιδήποτε άλλο, οι εικόνες έρχονται σε διαφορετικά είδη και εξυπηρετούν διαφορετικούς σκοπούς. Μερικές φορές, ένα μικρό μέγεθος είναι σημαντικό (για διαδικτυακούς τόπους) και άλλες φορές, η διατήρηση ενός υψηλού βάθους χρώματος σε όλο του το μεγαλείο (ένα οικογενειακό πορτρέτο) είναι αυτό που χρειαζόσαστε. Το GIMP μπορεί να χειριστεί όλα αυτά και περισσότερα, κυρίως μετατρέποντας μεταξύ τριών βασικών καταστάσεων, όπως φαίνεται σ' αυτό το μενού. Για να αλλάξετε την εικόνα σας σε μία από αυτές τις καταστάσεις, την ανοίγετε και ακολουθείτε το μενού πατώντας την κατάσταση που θέλετε.

Εικόνα 3.34. Διάλογος για αλλαγή κατάστασης


RGB - Αυτή είναι η προκαθορισμένη κατάσταση, χρησιμοποιούμενη για χρωματιστές φωτογραφίες πολύ υψηλής ποιότητας. Αυτή είναι επίσης η κατάσταση για το μεγαλύτερο μέρος εργασίας της εικόνας σας, συμπεριλαμβάνοντας κλιμάκωση, περικοπή, και ακόμα αναστροφή, καθώς δίνει στον υπολογιστή περισσότερες πληροφορίες για επεξεργασία. Οι επιπλέον πληροφορίες επίσης κάνουν την κατάσταση RGB τη πιο μεγάλη για αποθήκευση, ως συνέπεια.

Λίγες λεπτομέρειες εάν σας ενδιαφέρει. Κάθε εικονοστοιχείο ή σημείο, όταν βρίσκεται σ' αυτή την κατάσταση αποτελείται από τρία διαφορετικά συστατικά. R-> κόκκινο, G-> πράσινο, B-> μπλε. Καθένα από αυτά με τη σειρά του μπορεί να έχει τιμές έντασης 0-255. Έτσι, σε κάθε εικονοστοιχείο, που βλέπετε είναι ένας προσθετικός συνδυασμός αυτών των τριών συστατικών. Όλοι αυτοί οι συνδυασμοί καταλήγουν στην αντιπροσώπευση εκατομμυρίων χρωμάτων.

Από χρωματολόγιο - Αυτή η κατάσταση συνήθως χρησιμοποιείται όταν ενδιαφέρει το μέγεθος του αρχείου ή όταν εργάζεσθε με εικόνες με λίγα χρώματα. Εμπεριέχει ένα καθορισμένο αριθμό χρωμάτων, 256 ή λιγότερα, σε κάθε σημείο για απεικόνιση τού χρώματος σε αυτό το σημείο. Το καλύτερο είναι να επεξεργαστείτε μια "άριστη" παλέτα για καλύτερη απεικόνιση της εικόνας σας. Προσπαθήστε το, μπορείτε να το αναιρέσετε εάν δεν σας αρέσει το αποτέλεσμα, ή χρησιμοποιείστε μια προσαρμοσμένη παλέτα ή πιο πολλά χρώματα.

Εικόνα 3.35. Διάλογος “Αλλαγή χρωμάτων του χρωματολογίου”


Όπως θα περιμένατε, αφού η απαιτούμενη πληροφορία για απεικόνιση του χρώματος σε κάθε εικονοστοιχείο είναι πιο μικρή, το μέγεθος του αρχείου είναι πιο μικρό. Όμως μερικές φορές, υπάρχουν επιλογές στα ποικίλα μενού που είναι απενεργοποιημένες χωρίς κανένα προφανή λόγο. Αυτό συνήθως σημαίνει ότι το φίλτρο ή η επιλογή δεν μπορεί να εφαρμοστεί όταν η εικόνα σας είναι στην τρέχουσα κατάσταση. Αλλαγή της κατάστασης σε RGB, όπως περιγράφηκε πιο πάνω, θα πρέπει να επιλύσει το πρόβλημα. Εάν ούτε η κατάσταση RGB δεν δουλεύει, ίσως η επιλογή που δοκιμάζετε απαιτεί στρώση που μπορεί να είναι διαφανής. Αυτό μπορεί να αναιρεθεί εύκολα με Στρώση → Διαφάνεια → Προσθήκη καναλιού Alpha.

Εικόνα 3.36. Προσθήκη καναλιού Alpha


Κλίμακα του γκρι - Σε περίπτωση που θέλετε να μετατρέψετε την λαμπρή χρωματιστή εικόνα σας κάτι ασπρόμαυρο με πολλές αποχρώσεις του γκρι, αυτός είναι από τους πιο εύκολους τρόπους για να το κάνετε. Μερικές φωτογραφίες δείχνουν πιο φανταχτερές όταν εμφανίζονται στην κλίμακα του γκρι. Πιο λεπτομερειακά, αυτό πετυχαίνεται λαμβάνοντας τις τιμές RGB στα εικονοστοιχεία της εικόνας σας και παίρνοντας τους κατάλληλους μέσους όρους για την ένταση σε αυτό το σημείο.

Δεν χρειάζεται να μετατρέψετε μια εικόνα σε μια ειδική μορφή πριν την αποθηκεύσετε στην προτιμώμενη σας μορφή καθώς το GIMP είναι αρκετά έξυπνο ώστε να εξαγάγει κατάλληλα την εικόνα.
