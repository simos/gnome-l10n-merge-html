2.2. Επιπρόσθετη μείωση του μεγέθους αρχείου

Εάν θέλετε να μειώσετε το μέγεθος της εικόνας σας παραπέρα, θα πρέπει να μετατρέψετε την εικόνα σας σε κατάσταση από χρωματολόγιο. Αυτό σημαίνει ότι όλα τα χρώματα θα μειωθούν σε μόνο 256 τιμές. Μη μετατρέπετε εικόνες με ήπια μετάβαση χρωμάτων ή διαβαθμίσεις σε κατάσταση από χρωματολόγιο, επειδή οι αρχικές ήπιες διαβαθμίσεις της αρχικής εικόνας μετατρέπονται σε μια σειρά από λουρίδες. Η κατάσταση από χρωματολόγιο δε συνιστάται για φωτογραφίες, επειδή μετά τη μετατροπή φαίνονται υποβαθμισμένες και κοκκώδης.

Εικόνα 6.7. Εικόνα από χρωματολόγιο

Μια εικόνα από χρωματολόγιο μπορεί να φαίνεται λίγο κοκκώδης. Η αριστερή εικόνα του Wilber είναι στο αρχικό μέγεθος, η δεξιά εικόνα είναι μεγεθυμένη 300 τοις εκατό.


  1. Χρησιμοποιείστε την εντολή που περιγράφεται στο Ενότητα 6.3, “Κατάσταση” για να μετατρέψετε μια εικόνα RGB σε κατάσταση από χρωματολόγιο.

  2. Μετά τη μετατροπή σε εικόνα από χρωματολόγιο, μπορείτε πάλι να κάνετε αποθήκευση της εικόνας σε μορφή PNG.
